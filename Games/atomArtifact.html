How an Atom is Built

Atomic Number (Z) = number of protons = number of electrons in a neutral atom.

Mass Number (A) = protons + neutrons.

Electron Shells fill up in order: K-shell (2), L-shell (8), M-shell (8 for these elements), then a new shell starts.

Ions: a positive ion (cation) has lost electrons (fewer than protons). A negative ion (anion) has gained electrons (more than protons).

Use the + / – buttons to set each particle count, then tap Check Atom. Wrong tries show you whether to add or remove particles — no penalty for trying again!
